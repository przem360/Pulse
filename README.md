# Pulse

Pulse is an experimental piece, created entirely in the GNU/Linux terminal - written in lilypond, synthesised by TiMidity++ with VSCO  2:  Community  Edition soundfont.

[Listen on Soundcloud](https://soundcloud.com/pwolski/pulse)

## Tools used

 - [Lilypond](http://lilypond.org/)
 - [Lilypond bindings for vim (and Micro-Editor)](https://github.com/przem360/lilypond-bindings)
 - [TiMidity++](https://sourceforge.net/projects/timidity/)
 - [VSCO  2:  Community  Edition](https://vis.versilstudios.com/vsco-community.html)
 - [Green](https://github.com/schandinat/green)
